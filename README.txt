Contexto do Projeto:

Este projeto foi realizado como parte das atividades da disciplina de Lógica e Estrutura de Dados, fundamentado no livro “Entendendo Algoritmos” de Aditya Bhargava. 
O intuito foi explorar, tanto de maneira prática quanto teórica, o funcionamento dos sistemas de recomendação empregados em plataformas como Netflix, Spotify e Amazon,
utilizando conceitos de matemática, lógica e análise de dados. 


Objetivo

O propósito central deste projeto é ilustrar a criação de um sistema de recomendação de filmes que se baseia nas preferências dos usuários, por meio de:

- Análise de similaridade entre perfis de usuários;
- Aplicação de cálculos matemáticos para avaliar a proximidade entre esses perfis;
- Uso da média ponderada das avaliações para indicar filmes que o usuário ainda não assistiu.

Em síntese, a meta é descobrir quais filmes podem ser do agrado de um usuário, mesmo que ele não tenha manifestado isso de forma direta. 

Projeto:

Cada usuário é representado como um ponto em um gráfico, onde:

Usuários com interesses semelhantes estão próximos uns dos outros;

Usuários com gostos divergentes se encontram mais afastados.

Assim, se um usuário aprecia um filme, o sistema pode sugerir essa mesma obra a outro usuário que possua um perfil similar.

Exemplo:

Jonas e Tony compartilham gostos cinematográficos, portanto, se Tony der uma avaliação positiva a um filme que Jonas ainda não viu, o sistema poderá recomendá-lo a Jonas.

Por outro lado, usuários com preferências distintas, como Sophia, receberão sugestões diferentes.
